# AuNP_Mie_TMM_Simulation
This work presents a semi-analytical simulation framework combining Mie scattering theory and Transfer Matrix Method (TMM) to optimise an etched Corning single-mode (SMF) 28 optical fiber decorated with gold nanoparticles (AuNPs) for localized surface plasmon resonance (LSPR) based detection of cadmium, lead and mercury ions in aqueous medium.
